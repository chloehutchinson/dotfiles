# Dotfiles

## Dependencies
- iterm2 and a patched font

## Useful commands
`tmux new-session -A -s main`

## Installation
```sh
ln -s ~/dotfiles/config/zshrc ~/.zshrc
zsh
fresh
```

### AWS-CLI
For V2:
```sh
curl "https://d1vvhvl2y92vvt.cloudfront.net/awscli-exe-macos.zip" -o "awscliv2.zip"
unzip awscliv2.zip
sudo ./aws/install
```
