# Dotfiles

## Dependencies
- A patched font (e.g. Nerd Fonts)
- [fresh](https://github.com/freshshell/fresh)

## Useful commands
`tmux new-session -A -s main`

## Installation
```sh
ln -s ~/dotfiles/config/zshrc ~/.zshrc
zsh
fresh
```
