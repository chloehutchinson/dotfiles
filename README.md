# Dotfiles

## Dependencies
- iterm2 and a patched font

## Installation
```sh
ln -s ~/dotfiles/config/zshrc ~/.zshrc
zsh
fresh
```

## nvim configuration
1. Run `:PlugInstall` to install vim-plug plugins
1. Add plugins as needed for languages:
```
:CocInstall coc-tsserver coc-json coc-snippets
```
